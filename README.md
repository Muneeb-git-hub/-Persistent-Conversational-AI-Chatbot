# -Persistent-Conversational-AI-Chatbot with custom rag
 Developed a multi-session conversational agent using the Google Gemini API and LangGraph, integrating a custom 
Retrieval-Augmented Generation (RAG) pipeline for dynamic document Q&A. 
 Implemented robust tool-calling capabilities, enabling the agent to autonomously process user-uploaded documents and 
extract context-aware information in real time. 
 Engineered a reliable SQLite-based session management system with state-tracking workflows to maintain persistent 
memory across isolated user interactions.
