# -Persistent-Conversational-AI-Chatbot
 Built a persistent, multi-session conversational agent leveraging the Google Gemini API and LangGraph's state graph 
architecture. 
 Engineered a reliable session management system using SQLite, applying core concepts of memory persistence to retain 
user context over time. 
 Designed an efficient state-tracking workflow, ensuring smooth, context-aware interactions across isolated user sessions
